Several turbulence models are available, from Reynolds-averaged models to large-eddy simulation (LES) models. Running parallel Code Saturne jobsÂ¶

Ccde Saturne can exploit multiple nodes on ARCHER2 and will generally be run in exclusive mode.

For example, the following script will run a Code Saturne job using 4 nodes (128x4 cores).

Warning

The following SLURM script requires validation

#!/bin/bash

# Request 4 nodes with 128 MPI tasks per node for 20 minutes
# Replace [budget code] below with your account code,
# e.g. '--account=t01'

#SBATCH --job-name=CodeSaturne
#SBATCH --nodes=4
#SBATCH --tasks-per-node=128
#SBATCH --cpus-per-task=1
#SBATCH --time=00:20:00

#SBATCH --account=[budget code]

# Load the Code Saturne mdoule and run

module load code-saturne

code_saturne run --param case.xml
